## User Information Collected

The following attributes are to be captured from the user:

- **Age**  
  Used to evaluate age-based eligibility criteria for schemes.

- **Annual Family Income**  
  Acts as a primary factor for determining scheme eligibility and priority.

- **Location (State and District)**  
  Helps identify schemes applicable at central, state, or local levels.

- **Social Category**  
  Supports category-based welfare schemes and reservation benefits.

- **Gender**  
  Enables recommendations for gender-specific schemes where applicable.

---

## Usage in Recommendation Engine
User profile data is passed to the recommendation engine, where it is
matched against scheme eligibility rules. Each attribute contributes
to eligibility scoring and ranking in a transparent and explainable manner.

---

## Data Handling Notes
- User data is used only during the recommendation process
- No personal information is stored permanently
- Input validation is applied to ensure consistency and accuracy

---

## Summary
A well-defined user profile allows SchemeAssist AI to deliver accurate,
relevant, and explainable recommendations while maintaining simplicity
and ease of use.
